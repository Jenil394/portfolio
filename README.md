# Jenil Mansuriya — Portfolio

Personal portfolio website showcasing projects, skills, and achievements in machine learning, systems engineering, and developer tools.

## Tech Stack

- **Framework**: React + TypeScript (Vite)
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion + GSAP
- **UI Components**: shadcn/ui (Radix primitives)

## Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:8080`.

## Deployment

This project is deployed on **Vercel**. Push to the `main` branch to trigger a production deploy.

## Project Structure

```
src/
├── components/   # Reusable UI components
├── data/         # Portfolio content (projects, personal info)
├── pages/        # Route-level page components
├── types/        # TypeScript interfaces
└── main.tsx      # App entry point
```
